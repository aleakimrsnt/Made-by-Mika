Mobile Application Projects

HealthQuest – Gamified Wellness Tracker App

React Native, Firebase

A mobile application built with React Native and powered by Firebase on the backend, designed to promote healthier dietary habits through gamification and community engagement. It provides users with an interactive platform that makes adopting and maintaining healthier lifestyles more enjoyable. By combining dynamic features with a supportive community-driven approach, the application emphasizes sustainability, motivation, and long-term wellness.

Scantastic Eats – Recipe Recommendation App

Flutter Framework, Firebase, YOLOv8

A Flutter application with a Firebase backend designed to help university students and parents prepare lunch boxes more efficiently, reducing the time spent on meal planning and decision-making. It streamlines the process by offering practical features that simplify daily meal preparation. Enhanced with YOLOv8 technology for food recognition, it provides intelligent assistance in identifying ingredients and suggesting balanced meal options.

CoPartner – Rule-Based Chatbot System App

Flutter Framework, Supabase

A mobile application developed with the Flutter framework and powered by Supabase, designed to provide efficient and structured chatbot interactions. The system operates on rule-based logic, ensuring accurate and predictable responses tailored to user queries. With a clear and user-friendly interface, the app demonstrates how conversational support can be delivered in a reliable and accessible way.
